# ImageBeagle
Github for ImageBeagle dataset and search algorithm (Currently in progress). This is the code accompanying our paper [A tale of two tails: Preferred and anti-preferred natural stimuli in visual cortex](https://openreview.net/pdf?id=RZ8esDBqMJ) [1].


## Required Packages
```
- python 3.11.5
- torch 2.2.1
- keras 3.1.1
- tensorflow 2.16.1
- numpy 1.26.4
- matplotlib 3.8.3
- scipy 1.13.0
- scikit-learn 1.4.1
```
